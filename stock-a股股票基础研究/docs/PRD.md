# 股票一致预期监控系统 PRD

## 1. 项目概述

监测券商（卖方）研报的一致预期变化，通过时间序列分析判断市场预期走势，结合实际业绩数据判断超预期/低于预期。

## 2. 核心功能

### 2.0 核心价值：超/低预期即时提醒

**用户痛点**：业绩预告/公告发布后，需要人工逐个对比一致预期，耗时且容易遗漏

**解决方案**：系统自动监控 → 自动对比 → 第一时间推送

```
┌─────────────────────────────────────────────────────────┐
│                    自动化工作流                          │
│                                                         │
│  ① 数据采集                                             │
│     业绩预告(forecast) ──┐                              │
│     财报数据(fina)    ───┼──→ 实时监控新发布            │
│     券商预测(report_rc) ─┘                              │
│                          │                              │
│  ② 自动对比              ▼                              │
│     ┌────────────────────────────────────┐             │
│     │ 业绩预告/实际业绩  vs  一致预期     │             │
│     │ (公司公告)            (券商中位数)  │             │
│     └────────────────────────────────────┘             │
│                          │                              │
│  ③ 筛选符合条件的标的    ▼                              │
│     ┌────────────────────────────────────┐             │
│     │ 超预期 > +X%  │  低于预期 < -X%    │             │
│     └────────────────────────────────────┘             │
│                          │                              │
│  ④ 即时推送              ▼                              │
│     ┌────────────────────────────────────┐             │
│     │ Telegram 消息推送                   │             │
│     │ 包含：股票、预期差、关键数据        │             │
│     └────────────────────────────────────┘             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.1 数据采集模块

| 数据源 | 接口 | 说明 | 优先级 |
|--------|------|------|--------|
| 券商盈利预测 | `report_rc` | 卖方研报的EPS/PE等预测数据 | P0 |
| 业绩预告 | `forecast` | 公司发布的业绩预告 | P1 |
| 财务报告 | `fina_indicator` | 实际财报数据 | P1 |

**当前进度**：`report_rc` 脚本已完成 (`scripts/fetch_report_rc.py`)

### 2.2 数据存储

| 格式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| CSV | 简单、可读 | 查询慢、无索引 | 原始数据备份 |
| SQLite | 支持SQL查询、轻量 | 并发写入受限 | 本地分析首选 |
| HDF5 | 高效存储大数据、支持压缩 | 需要专门工具读取 | 量化回测场景 |

**建议**：采用 **SQLite** 作为主存储，CSV 作为导出格式

### 2.3 预期变化分析

**原始数据结构** (report_rc):

| 字段 | 说明 | 示例 |
|------|------|------|
| ts_code | 股票代码 | 600436.SH |
| name | 股票名称 | 片仔癀 |
| report_date | 研报日期 | 20260130 |
| report_title | 报告标题 | 控股股东增持... |
| report_type | 报告类型 | 点评/深度 |
| classify | 报告分类 | 一般报告/首次关注 |
| org_name | 机构名称 | 中金 |
| author_name | 作者 | 朱言音,夏璐 |
| quarter | 预测报告期 | 2025Q4/2026Q4/2027Q4 |
| op_rt | 预测营业收入(万元) | 968700.0 |
| np | 预测净利润(万元) | 243100.0 |
| eps | 预测每股收益(元) | 4.03 |
| pe | 预测市盈率 | 41.4 |
| roe | 预测净资产收益率 | 16.7 |
| rating | 卖方评级 | 跑赢行业/买入/增持 |
| min_price | 预测目标价 | 240.0 |

**数据特点**：
- 同一份研报包含多条记录，对应不同预测年份（2025Q4/2026Q4/2027Q4）
- 同一天可能有多家券商对同一公司发布研报
- 需按 `ts_code + quarter` 分组，按 `report_date` 追踪变化

**时间序列示例**（追踪同一公司同一预测年份的预期变化）:
```
片仔癀(600436.SH) - 2026Q4预测:
┌──────────┬──────────┬────────┬────────┬────────┐
│ 报告日期  │ 机构     │  EPS   │ 净利润 │ 评级   │
├──────────┼──────────┼────────┼────────┼────────┤
│ 20260115 │ 国泰海通 │  4.20  │ 25.3亿 │ 增持   │
│ 20260122 │ 招商证券 │  4.35  │ 26.1亿 │ 买入   │ ← 预期上调
│ 20260130 │ 中金     │  4.43  │ 26.7亿 │ 跑赢行业│ ← 继续上调
└──────────┴──────────┴────────┴────────┴────────┘
```

**监控指标**：
- 新增覆盖公司（classify=首次关注/首次评级）
- EPS/净利润预期变化（同公司同预测年份，不同日期对比）
- 评级变化（买入→持有等）
- 目标价变化

### 2.4 核心监控场景

#### 场景A：业绩预告 vs 一致预期（即时触发）

```
┌─────────────────────────────────────────────────────────┐
│ 输入                                                     │
│  - 市场一致预期：某公司26年净利润 12亿（券商平均/中位数）│
│  - 公司发布业绩预告：净利润 15-16亿                      │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│ 判断                                                     │
│  预告下限(15亿) > 一致预期(12亿) → 超预期 +25%          │
│  预告上限(16亿) > 一致预期(12亿) → 超预期 +33%          │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│ 触发 Agent 分析                                          │
│  - 超预期原因：行业景气？成本下降？新业务放量？          │
│  - 可持续性判断                                          │
│  - 后续预期是否需要上调                                  │
└─────────────────────────────────────────────────────────┘
```

**触发条件**：
| 情况 | 条件 | 动作 |
|------|------|------|
| 超预期 | 预告下限 > 一致预期 × (1 + 阈值) | Agent 分析超预期原因 |
| 低于预期 | 预告上限 < 一致预期 × (1 - 阈值) | Agent 分析低于预期原因 |
| 符合预期 | 预告区间覆盖一致预期 | 正常记录，不触发 |

#### 场景B：公告后的卖方预期调整监控（延时追踪）

```
时间线：
Day 0: 公司发布业绩预告/公告
Day 1-2: 卖方陆续发布点评报告（存在时差）
Day 3+: 预期调整基本完成

┌─────────────────────────────────────────────────────────┐
│ 监控窗口（公告后 1-5 天）                                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  公告前一致预期（26年EPS）                               │
│  ┌─────────────────────────────────────┐               │
│  │ 中金: 4.20  招商: 4.35  广发: 4.10  │ → 中位数 4.20  │
│  └─────────────────────────────────────┘               │
│                    │                                    │
│                    ▼ 公司发布超预期公告                  │
│                                                         │
│  公告后一致预期（26年EPS）                               │
│  ┌─────────────────────────────────────┐               │
│  │ 中金: 4.60  招商: 4.80  广发: 4.50  │ → 中位数 4.60  │
│  └─────────────────────────────────────┘               │
│                                                         │
│  变化量化：                                              │
│  - 中位数变化: 4.60 - 4.20 = +0.40 (+9.5%)             │
│  - 单家变化: 中金 +0.40, 招商 +0.45, 广发 +0.40        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**需要量化的指标**：

| 指标 | 说明 |
|------|------|
| 预期变化幅度 | 公告后 vs 公告前的 EPS/净利润中位数变化 |
| 单家券商调整 | 每家券商的预期调整幅度 |
| 调整方向一致性 | 是否所有券商都同向调整 |
| 调整覆盖率 | 有多少比例的券商已更新报告 |
| 调整时效 | 公告发布到券商更新报告的平均时差 |

#### 场景C：预期趋势持续追踪

```
某公司26年EPS一致预期走势：
┌────────────────────────────────────────────────────────┐
│  EPS                                                    │
│  5.0 ┤                                    ○ 4.80       │
│  4.5 ┤                          ○ 4.60                 │
│  4.0 ┤          ○ 4.20   ○ 4.20                        │
│  3.5 ┤   ○ 3.80                                        │
│  3.0 ┤                                                  │
│      └──┬────────┬────────┬────────┬────────┬──────    │
│       01/10    01/15    01/20    01/25    01/30        │
│                                                         │
│  趋势判断: 持续上调 (+26% in 20 days)                   │
└────────────────────────────────────────────────────────┘
```

**趋势分析输出**：
- 方向：持续上调 / 持续下调 / 震荡
- 幅度：区间内累计变化百分比
- 加速度：近期调整是否加速
