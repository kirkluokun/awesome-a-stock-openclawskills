# 🤖 Awesome A-Stock OpenClaw Skills

精选的 OpenClaw / Claude 代理技能集合，专注于 A 股投资研究与分析。

## 什么是 OpenClaw Skills？

OpenClaw Skills 是基于 Markdown 的 AI 代理技能定义文件，可以扩展 Claude 等 AI 助手的能力，使其具备专业的金融分析、数据获取、研究协作等功能。
